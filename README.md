# James Alexander Portfolio SPA

Static single-page portfolio built with React + TypeScript + Vite.

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Production output is generated in `dist`.

## Cloudflare Pages setup

- Framework preset: `Vite`
- Build command: `npm run build`
- Build output directory: `dist`
- Node version: `20` or newer

SPA routing fallback is configured with `public/_redirects`.

## Resume asset

Downloadable resume is served from:
